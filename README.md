# Figma vers HTML

Figma vers HTML est une extension Figma qui vous permet d'exporter directement vos maquettes sous la forme de code HTML, de styles CSS et d'interactions JavaScript prêts à l'emploi. Ce projet est conçu pour être rapide, propre et facile à lire.

## Fonctionnalités

* Conversion Haute Fidélité : Prend en charge de manière native les Auto-Layouts Figma, la typographie, les espacements, les arrondis, et convertit tout en CSS moderne.
* Export SVG Intelligent : Les icones, vecteurs complexes ou opérations booléennes sont automatiquement extraits et insérés en tant que balises SVG proprement formatées dans le HTML.
* Variables et Images : Les images de fond sont encodées à la volée, de même pour les dégradés et ombres portées complexes.
* Prototypes Interactifs Intégrés : Le code extrait préserve les composants interactifs (boutons, interrupteurs, cases à cocher) via un parcours intelligent en arrière plan. Le clic et le survol sont gérés pour changer d'état, avec des transitions fluides qui reprennent exactement vos configurations Figma (linéaire, entrées/sorties progressives, courbes de Bézier personnalisées).
* Préservation des Textes : Les textes surchargés ou modifiés sur vos différentes instances sont conservés, même à travers des composants animés.

## Utilisation

1. Sélectionnez un élément.
2. Cliquez sur "Exporter en ZIP".
